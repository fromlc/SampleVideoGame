# Sample Video Game for CS 281

This code is derived from Dr. Churchill's COMP 4300 Assignment 2,
presented on YouTube: https://youtu.be/-ugbLQlw_VM?si=tZnH4CMx3jueOFG5

Dr. C's Entity Component System architecture for A2 is simple and
well-suited for intermediate C++ students. He includes instructive
features of C++, OOP, and ECS. He uses SFML v2.5.1, an easy
graphics library to produce 2D games with.

Dr. C only provides assignment starter code to enrolled students,
but his public video lectures are detailed enough to make A2 doable
from scratch, as here. 

This game is similar to A2 but simpler and it uses SFML v3.0.2.
This code retains Dr. C's ECS architecture and instructive 
features, but these modifications make it useless as A2 solution
code.
